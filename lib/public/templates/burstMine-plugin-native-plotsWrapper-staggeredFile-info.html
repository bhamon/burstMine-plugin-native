The staggeredFile plots wrapper uses a native binding to handle plots in a file with a parametric staggerSize value. It is a basic implementation of the original file writer (address_offset_number_staggerSize). The scoops are stored in a file and are partially optimized in large chunks. The chunks size is controlled by the staggerSize parameter.

The plots writers created from this wrapper use a buffer with a length of staggerSize plots to reorder the generated scoops. Once the buffer is full, it is written to disk.

The plots readers created from this wrapper generates plotsNumber / staggerSize input operations on the target disk.
